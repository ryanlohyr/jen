# MINDS MYG Design System

> **MINDS MYG Design System is currently in BETA**.

## Get started

- **Explore UI components**
  Click on any of the component in the sidebar to see how they look, and how to configure them. Each component pages contains documentation of its properties, as well as code examples for how to use it.
- **Try them out**
  - Experiment with different properties, by clicking **Canvas** at the top of each component page.
  - Change the properties under **Actions** in the bottom panel.
- **Use them**
  Once you've found the right component for your use case, look at examples under **Docs**, click on **Show Code** to copy the code implementation.
